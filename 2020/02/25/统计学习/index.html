第二章：统计学习

什么是统计学习？

统计学习就是使用一系列的方法去估计输入和输出之间的关系

也就是这里面的f

为什么估计f

我们基于两个主要的原因想要去估计f

  • 预测：我们关注的是预测的response的好坏，不关心f的具体形式，因此f可以是一个黑盒子；这里对Y的预测的精确度取决于两个量：

    这里的reducible是可以通过选择合适的统计方法去减少的，但是irreducible是不可人为减少的，这个一般包含不可测量的变量或者某个变量有不可测量的variation，比如：我们根据病人的血样来预测其对药物的不良反应程度，这个预测可能与该药本身的发挥作用的差异，或者病人的心情相关

  • 推断：我们关注的是Y是怎么随着X的变化而变化的，哪些变量与response相关的，如何识别比较重要的变量，这个时候我们就要知道f的准确的形式，而不是当作黑盒看待

怎么样去估计f

我们要基于训练数据依据某种统计学习的方法去估计f，统计学习的方法可以大致分成两类：参数和非参数的

  • 参数的方法：我们要先假设一个f的形式，再用训练数据来拟合我们假设的模型，估计模型中的参数，缺点是：依赖于我们选择的模型
  • 非参数的方法：不假设f的形式，直接用数据去估计f，缺点就是要求的数据量比较大，并且容易出现过拟合

在预测的精确度和模型的可解释性之间平衡

监督学习和非监督学习

还有一种叫做半监督学习，就是我们有n个观测值，其中m个有response，另n-m个没有response

回归和分类问题

一般来讲，对于定量的变量用回归，对于定性的用分类，但是也有很多例外，比如logistic regression 也可以用作定量变量的回归，还有boosting和k最近邻方法对定量和定性的变量都适用（这里的变量指的是response，也就是因变量）

评估模型精确度

拟合质量的检验

也就是我们要定量预测的response和真实的response之间差多少

对于回归来说，一般用均方误差来衡量（MSE）

对于统计学习有一个基本的特征：随着统计学习方法的自由度的上升，训练集的MSE会单调下降，而测试集的MSE会呈U型，先下降后上升；当一种方法在训练集中产生小的MSE，而在测试集中产生大的SSE，我们就说这种方法过拟合了数据

偏差-方差之间的平衡

expected test MSE的期望由3部分构成：

上式左边指的是expected test MSE，即我们在一个大的训练集中重复估计f，然后对每一个f在测试集的x0处计算其MSE，再取其平均值

variance指的是如果我们用一个不同的训练集（和之前的类似）来估计f，两次f的差距

bias指的是和实际情况的差距，比如实际情况是一个非线性的关系，我们用一个线性模型去拟合，那么bias就会比较大

一般来说，更自由的模型会导致variance上升，而bias下降

所以我们要找到偏差-方差之间的平衡，从而使得expected test MSE最小
